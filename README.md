# qt-multithreading

Исходный код для классов BlockingClient, ClientThread, FortuneServer и SereverThread взят из [официальныого репозитория Qt, ветки 5.15](https://github.com/qt/qtbase/tree/5.15) и модифицирован.

## Сборка и запуск
При помощи CMake собирается два исполняемых файла: app и tests -- необходимые для запуска приложения GUI и гугл-тестов соотсветственно. Запуск можно осуществить при помощи созданного спкциально для этого bash-скрипта.

### Из bash-скрипта
В корне проекта выполнить команды
1) Для выполнения тестов
```
bash start.sh -t
```
2) Для запуска приложения

```
bash start.sh -g
```
### Вручную
1. Удалить директорию build/, если она уже существует
```
rm -rf build
```
2. Создать build/ и запустить из нее сборку проекта
```
mkdir build
cd build
cmake ..
make
```

3. Запустить проект

a) для тестов
```
./tests
```
б) для GUI-приложения
```
./app
```
